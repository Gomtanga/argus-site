Features

Research that doesn't cut corners

Argus goes beyond single-query search with a systematic framework designed for depth, accuracy, and accountability.

4-Phase Search Framework

Broad exploration, targeted investigation, cross-verification, and comparative analysis — each query builds on the last.

Source Verification

Every major claim is backed by citations in [web:N] format. No unverified assertions, no single-source reliance.

Quantitative Data

Vague claims are rejected. Argus demands specific numbers — benchmarks, metrics, and measurable evidence.

Adaptive Complexity

Simple questions get quick answers. Deep investigations get up to 14+ searches across multiple phases until saturation.

Multi-Tool Integration

Leverages Firecrawl's full suite — search, scrape, agent, and map — for comprehensive web data extraction.

Graceful Degradation

When tools fail or rate limits hit, Argus adapts — reducing scope, using fallbacks, or honestly stating limitations.

Principles

What Argus will never do

Clear rules prevent common AI research failures.

✕

Assert facts without verification through search

✕

Make major claims based on a single source

✕

Present numerical data without citing a source

✕

Present outdated information as current

Get Started

Up and running in minutes

Argus works as an OpenCode agent skill. Install it, trigger it with natural language, and let it research.

1

Prerequisites

You need OpenCode and Firecrawl tools activated (firecrawl_search, firecrawl_scrape, firecrawl_agent, firecrawl_map).

2

Install the Skill

Place the skill file in your OpenCode skills directory. Argus auto-activates when it detects research intent.

3

Just Ask

Use natural language: "Compare React vs Vue performance in 2025" — Argus handles the rest.

Trigger Words

search find investigate compare analyze look up web search research
